React Notes

Getting Started:

What is React?

React was developed by Facebook in 2011. React is a frontend JavaScript library.

Basics:

Create react app by running "create-react-app [name]" from command line

View project in browser by running "npm start" from project directory

A skeleton react app will look like

The index.html file inside the public directory is a normal HTML file. You can change the favicon and title there.

Installing Bootstrap:

Run "npm i bootstrap" from project directory. This installs bootstrap in the node modules

Then, "import 'bootstrap/dist/css/bootstrap.css'" to index.js. This will import bootstrap from the node modules into the app
