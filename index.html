Starting a React Project

  • Install Node.js. This will give you access to Vite as Vite is an npm package
  • npm create vite@latest [project name]
  • Select React from list of libraries/frameworks (other options vue, svelte, etc.)
  • Select variant (TypeScript, TypeScript + speedy web compilier, JavaScript, JavaScript + speedy web compilier)
  • cd into [project name]
  • npm install to install dependencies
  • npm run dev to open project in browser

  • check react version with npm view react version
